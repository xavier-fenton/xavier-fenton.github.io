Describe what the DOM is and an example of how you might interact with it.

The Document Object Model, generally abbreviated to DOM, is the way through which JavaScript interacts with content within a website.

The DOM represents the HTML as a tree structure.



The Dom uses an interactive web interface to use commands from JavaScript. It has the ability to change the document structure, style and content. The DOM can be easily accessed on a webpage by anyone; through the inspect button when you right click a page. But with JavaScript we can add new things without hardcoding into our HTML, or CSS files. This would be considered manipulating the DOM. A brief example could be that if you have a blank HTML file; with JavaScript we could add content like a header and some paragraphs by interacting with the DOM without touching our HTML. Well it’s like another way of adding elements to your files.


But this has the potential of so many things. Say someone clicks a button that we’ve scripted that can randomly generate new content on our webpage. Like our title changes from “Hello World” to “Hello Everyone”, without touching the HTML file. Or the button changes the background colour to some random colour than the original. JavaScript allows much more to a webpage to be so interactive, or have a user-friendly flow to the page.

Explain what functions are and why they are helpful:

I see functions as being one of many tools to be a very important part in JS, with functions we basically are telling the computer to perform a task. Breaking down functions is useful.

Function declaration (naming a function):

function greetings (name){
console.log(‘Hello ‘ + name);
}

We call our functions with console.log to let it output “Hello name”. But with will only see "Hello". Because calling requires an argument which refers to the parentheses of our function.

greetings(‘Xavier’)

We should see an output of “Hello Xavier”. I tend to see functions and commands within JS as connecting the dots. For instance within the parentheses of our function declaration is name which is the parameter we are referring to, and our argument when calling our function is Xavier. This is helpful for reusing such a function we can add more parameters to our function declaration to make a long string like: “Hello Xavier Fenton”, functions allow us to make more logical use of code.

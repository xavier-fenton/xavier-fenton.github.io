Reflect on how confident you feel using each of these problem-solving techniques/processes:

Overall reflecting on the methods used for problem solving so far asking peers for help and the coaches has made my approach for programming much more confident about my future as becoming a developer. That there is always others that a willing to help, and I hope I can be counted on too in the future.

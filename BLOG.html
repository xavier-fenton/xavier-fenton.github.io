What makes a website?

A website is built up of 3 components/ languages that are: -HTML -this is the foundation of the webpage, the structure that builds it up. -CSS -This is the design aspect which makes everything look good -Javascript -The wizard that does all the crazy code, pop errors and what not. These 3 Languages make up what is called front-end web development -what you see when you land on a webpage Internet applications translate these 3 languages into the visual webpage. They are all constantly evolving.
